What is an Operator?

Let us take a simple expression 4 + 5 is equal to 9. Here 4 and 5 are called operands and ‘+’ is called the operator. JavaScript supports the following types of operators.

❖ Arithmetic Operators
❖ Comparison Operators
❖ Logical (or Relational) Operators
❖ Assignment Operators
❖ Conditional (or ternary) Operators

Let’s have a look at all the operators one by one.

Arithmetic Operators

JavaScript supports the following arithmetic operators:
Assume variable A holds 10 and variable B holds 20, then:

S.No. Operator and Discription
1. + (Addition)
Adds two operands
Ex: A + B will give 30
2. - (Subtraction)
Subtracts the second operand from the first
Ex: A - B will give -10
3. * (Multiplication)
Multiply both operands
Ex: A * B will give 200
4. / (Division)
Divide the Numerator by the Denaminator
Ex: A / B will give 2
5. %
Outputs of the Remainder of an integer dividion
Ex: A % B will give 0
6. ++ (Increment)
Increases an integer value by one
Ex: A++ will give 11
7. -- (Decrement)
Decrease an integer value by one
Ex: A-- will give 9


Note: Addition operator (+) works for Numeric as well as Strings. e.g. "a" + 10 will give "a10".

Bitwise Operators

JavaScript supports the following bitwise operators:

Assume variable A holds 2 and variable B holds 3, then:

S.No. Operator and Discription
1. & (Bitwise AND)
Its performs a Boolean AND operation on each bit its integer arguments.
Ex: (A & B) is 2
2. | (Bitwise OR)
ts performs a Boolean OR operation on each bit its integer arguments.
Ex: (A | B) is 3
3. ^ (Bitwise XOR)
It performs a Boolean exclusive OR operation on each bit of its integer
arguments. Exclusive OR means that either operand one is true or operand
two is true, both not both.
Ex: (A ^ B) is 1
4. ~ (Bitwise Not)
It is a unary operator and operates by reserving all the bits in the operand.
Ex: (~B) is -4
5. << (Left Shift)
It moves all the bits in its first operand to the left by the number of places
specified in the second operand. New bits are filled with zeros. Shifting a
value left by one position is equivalent to multiplying it by 2, shifting 2
positions is equivalent to multiplying by 4, and so on.
Ex: (A << B) is 4
6. >> (Right Shift)
Binary Right Shift Operator. The left operand's value is moved right by the
number of bits specified by the right operand.
Ex: (A >> B) is 1
7. >>> (Right Shift with Zero)
This operator is just like the >> operator, except the bills shifted in
on the left are always zero.
Ex: (A >>> 1) is 1


Example

Try the following code to implement Bitwise operator in JavaScript.

Assignment Operators

JavaScript supports the following assignment operators:

S.No. Operator and Discription
1. = (Simple Assignment)
Assigns values from the right side operand to the left side operand
Ex: C = A + B will assign the value of A + B into C
2. += (Add and Assignment)
Its adds the right operand to the left operand and assigns the result to the left operand.
Ex: C += A is equivalent to C = C + A
3. -= (Subtract and Assignment)
Its subtract the right operand to the left operand and assigns the result to the left operand.
Ex: C -= A is equivalent to C = C - A
4. *= (Multiply and Assignment)
Its multiply the right operand to the left operand and assigns the result to the left operand.
Ex: C *= A is equivalent to C = C * A
5. /= (Divide and Assignment)
Its divide the right operand to the left operand and assigns the result to the left operand.
Ex: C /= A is equivalent to C = C / A
6. %= (Modulus and Assignment)
It take modulus using two operands and assigns the result to the left operand
Ex: C %= A is equivalent to C = C % A


Note: Same logic applies to Bitwise operators, so they will become <<=, >>=, >>=, &=, |= and ^=.

S.No. Operator and Discription

typeof Operator

The typeof operator is a unary operator that is placed before its single operand, which can be of any type. Its value is a string indicating the data type of the operand.

The typeof operator evaluates to "number", "string", or "boolean" if its operand is a number, string, or boolean value and returns true or false based on the evaluation.

Here is a list of the return values for the typeof Operator.

Type String Returned by typeof
Number "number"
String "string"
Boolean "boolean"
Object "object"
Function "function"
Undefined "undefined"
Null "null"
