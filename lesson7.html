What is a Popup Boxes?

Popup boxes are those windows that pop-out of your screen. They could be before a page is fully loaded, effect of an event on a page or before leaving a page.

JavaScript has three kinds of popup boxes and these are

❖ Alert box
❖ Prompt box
❖ Confirm box

These popup boxes can be useful in warning the user, gathering user input or simply to verify something.

ALERT BOX syntax alert(“message”);
PROMPT BOX syntax prompt(“message”, “defaultvalue”);
CONFIRM BOX syntax confirm(“message”);


Example of Alert Box

<!DOCTYPE html> <html> <head> <script type="text/JavaScript"> function show_alert() { alert("WARNING! WARNING!"); } </script> </head> <body> <input type="button" onclick="show_alert()" value="show alert box" /> </body> </html>




Example of Prompt Box

<!DOCTYPE html> <html> <head> <script type="text/JavaScript"> function show_prompt() { var name=prompt("Please enter your name: ", "nameless"); if (name!=null && name!="") { document.write("Welcome " + name + "! Please come in" ); } } </script> </head> <body> <input type="button" onclick="show_prompt()" value="show prompt box" /> </body> </html>




Example of Confirm Box

<!DOCTYPE html> <html> <head> <script type="text/JavaScript"> function show_confirm() { var ans=confirm("Do you want to proceed? "); if (ans==true) { document.write("Access granted."); } else { document.write("Access denied."); } } </script> </head> <body> <input type="button" onclick="show_confirm()" value="Show confirm box" /> </body> </html>
